EXPERIMENT 3.1

Development: Pushing visual quality using Blender simulations. Moving beyond simple geometry to particle physics.

Visual Goal: Performers want high-quality, abstract aesthetics. The goal was strong visual impact without literal interpretation.

Technology: Utilizing Blender's EEVEE engine for real-time, complex visuals essential for responsive performance.

VARIATIONS

EXPERIMENT 3.2

Process: Connected an industry-standard MIDI controller to Resolume Arena to test current professional tools.

Observations: The Resolume interface is complex. The sheer number of parameters highlights the immense cognitive load placed on the user during live performance.

Critique: While functional, standard MIDI lacks specificity. Its simple binary input cannot meet the nuanced control requirements of cultural percussion. The project must build a custom tool.
