Description

This project explores iterations of interfaces as a way to build a visual toolkit through making. The focus is on experimenting with different approaches to visual creation, using critical making and critical journaling as methods of reflection and exploration. By refining the toolkit, the project explores on how the experiments are able to contribute to the toolkit.
